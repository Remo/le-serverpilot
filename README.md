# le-serverpilot
SH script to install / manage Lets Encrypt for Server Pilot free users

#** PLEASE USE AT YOUR OWN RISK **

##Requirements

> Ubuntu 14.04 
> Server running with Serverpilot
> Root User Access

##How to Install

```
cd le-df
chmod +x df.sh
```

##How to Use

```
cd le-df
./df.sh
```
